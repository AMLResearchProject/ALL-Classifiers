# Acute Myeloid/Lymphoblastic Leukemia Pure Python Classifiers
![Peter Moss Acute Myeloid/Lymphoblastic Leukemia Detection System](../../Media/Images/banner.png)
The Peter Moss Acute Myeloid/Lymphoblastic Leukemia Pure Python classifiers are a collection of projects that use computer vision written in Python to classify Acute Myeloid/Lymphoblastic Leukemia in unseen images.
